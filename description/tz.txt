После того, как все тесты пройдены, у профориентатора становится активной вкладка Профориентация.
Блок «Сферы»
Профориентатор может либо 
	•	Выбрать сферы из 12 (указаны в spheres.json) в режиме топ-1, топ-2, топ-3 (порядок важен), либо
	•	Нажать на кнопку «Получить рекомендацию от ИИ»
При нажатии на кнопку отправляется запрос, результатом которого является 3 отранжированные отрасли (см. Выбор отраслей) + комментарии, почему они подходят
При работе с этим блоком у профориентатора есть возможность отредактировать сферы (аналогично «скорректировать» у внеурочки)
После того, как сферы выбраны, активируется кнопка «сгенерировать отчёт» (см. ниже).
Кнопка генерации отчёта есть только у профориентатора, после того, как отчёт сгенерирован, во вкладке у школьника появляется веб-версия отчёта (см. ниже) и ссылка на скачивание PDF

Сгенерировать отчёт
Определение профессий
	•	Подготовка данных. Из полных результатов по RIASEC и софтам готовятся нормализованные вектора (0..100), по High5 используется полный вектор
	•	Данные упаковываются в общий вектор размерности 6+6+10=22ВНИМАНИЕ НА ПОРЯДОК: ["R", "I", "A", "S", "E", "C"], ["COM", "TEAM", "CRE", "SYS", "EQ", "UNC"], ["Thinker", "Problem Solver", "Strategist", "Philomath", "Coach", "Empathizer", "Storyteller", "Catalyst", "Believer", "Self-believer",]
	•	По выбранным сферам собираются 2 (!) аналогичных вектора для профессий из professions_full.json: для профессий уровня "education_level": "TVET" и для "education_level": "Higher"
	•	Вектор пользователя сравнивается (косинусное сходство) со всеми векторами профессий, и по каждой из сфер отбираются топ-3 наиболее близкие. Это делается отдельно для профессий уровня TVET (колледж) и Higher (вуз)
	•	Итого, результатом является набор из 3х3 профессий для высшего образования и 3х3 профессий для уровня колледжа
