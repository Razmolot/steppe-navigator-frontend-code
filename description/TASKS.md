# Steppe Navigator: Профориентационный модуль — Список задач

## Общее описание

Модуль включает два типа отчётов:
1. **Профориентационный отчёт ученика** — индивидуальный отчёт с результатами тестов, сферами и профессиями
2. **Отчёт о профориентационной работе** — агрегированный отчёт по классу/параллели/школе

После прохождения всех тестов у профориентатора активируется вкладка **«Профориентация»**.

## 1. Блок «Сферы»

### 1.1 Интерфейс выбора сфер
- [ ] Реализовать UI для выбора сфер из 12 доступных (данные в `prompt_spheres.json`)
- [ ] Реализовать режим топ-1, топ-2, топ-3 с сохранением порядка
- [ ] Реализовать кнопку «Получить рекомендацию от ИИ»
- [ ] Реализовать возможность редактирования/корректировки выбранных сфер

### 1.2 API для рекомендации сфер от ИИ
- [ ] Создать endpoint для получения ИИ-рекомендации сфер
- [ ] Собирать входные данные:
  - Топ-3 RIASEC-код (например, "ERI")
  - JSON ответов пользователя на общую часть теста на английском
  - Комментарий профориентатора (переводить на английский перед отправкой)
- [ ] Интегрировать OpenAI API с промптом из `prompt_spheres.json`
- [ ] Парсить ответ LLM в формате:
  ```json
  {
    "top_fields": [
      {
        "id": "SPHERE_ID",
        "name_en": "Sphere Name",
        "score": 0.85,
        "reasoning_en": "Explanation"
      }
    ]
  }
  ```

### 1.3 Перевод результатов ИИ
- [ ] Реализовать функцию перевода через OpenAI (логика в `translate.py`)
- [ ] Генерировать `reasoning_ru` и `reasoning_kk` для отображения на фронте

## 2. Определение профессий

### 2.1 Подготовка данных пользователя
- [ ] Нормализовать результаты RIASEC в вектор (0..100) размерности 6
- [ ] Нормализовать результаты Soft Skills в вектор (0..100) размерности 6
- [ ] Использовать полный вектор High5 размерности 10
- [ ] Объединить в общий вектор размерности 22

**Порядок элементов вектора:**
- RIASEC: `["R", "I", "A", "S", "E", "C"]`
- Soft Skills: `["COM", "TEAM", "CRE", "SYS", "EQ", "UNC"]`
- High5: `["Thinker", "Problem Solver", "Strategist", "Philomath", "Coach", "Empathizer", "Storyteller", "Catalyst", "Believer", "Self-believer"]`

### 2.2 Алгоритм подбора профессий
- [ ] Загрузить профессии из `professions_full.json` по выбранным сферам
- [ ] Разделить профессии по уровню образования: `TVET` (колледж) и `Higher` (вуз)
- [ ] Для каждой профессии собрать аналогичный 22-мерный вектор
- [ ] Вычислить косинусное сходство между вектором пользователя и векторами профессий
- [ ] Отобрать топ-3 профессии по каждой из 3 сфер отдельно для TVET и Higher
- [ ] Итоговый результат: 3×3 профессий для вуза + 3×3 профессий для колледжа

## 9. Профориентационный срез (диаграммы)

Данные берутся из ответов учеников на общий тест. В легендах указывается **процент** (не количество).

### 9.1 Диаграмма «Планы после школы»
- [ ] Тип: Pie Chart
- [ ] Вопрос: «Чем вы собираетесь заняться после окончания школы?»
- [ ] Варианты ответов:
  - После 11 класса поступить в университет
  - После 9 класса поступить в колледж
  - После школы планирую работать
  - После школы планирую служить в армии
  - После школы возьму академический отпуск
  - Затрудняюсь ответить

### 9.2 Диаграмма «Где планируют учиться»
- [ ] Тип: Pie Chart
- [ ] Вопрос: «Где вы планируете продолжить обучение?»
- [ ] Варианты ответов:
  - В области, где живу сейчас
  - В другом регионе Казахстана
  - В другой стране
  - Затрудняюсь ответить

### 9.3 Диаграмма «Критерии выбора учебного заведения»
- [ ] Тип: Pie Chart
- [ ] Вопрос: «Что для вас важно при выборе учебного заведения?»
- [ ] Логика: **множественные ответы** — складываем и считаем процент
- [ ] Варианты ответов:
  - Мои способности и интересы
  - Рейтинг учебного заведения
  - Количество грантов по специальности
  - Регион учебного заведения
  - Затрудняюсь ответить

### 9.4 Диаграмма «Определились с профессией»
- [ ] Тип: Bar Chart (вертикальный)
- [ ] Вопрос: «Вы уже определились с будущей профессией?»
- [ ] Варианты ответов:
  - Да, выбрал один вариант, стремлюсь к нему
  - Выбрал несколько возможных вариантов, определяюсь
  - Не определился

### 9.5 Диаграмма «Чьё мнение важно»
- [ ] Тип: Horizontal Bar Chart
- [ ] Вопрос: «Чьё мнение важно для вас при выборе профессии?»
- [ ] Логика: **множественные ответы** — аналогично п.9.3
- [ ] Варианты ответов (ранжирование):
  - Собственное
  - Родителей и близких
  - Представителей профессии
  - Друзей и знакомых
  - Учителей
  - Известных людей, которым я доверяю

### 9.6 Диаграмма «Рекомендованные сферы»
- [ ] Тип: Pie Chart
- [ ] Логика: берём **все 3 предложенные сферы** каждого ученика, считаем процент
- [ ] 12 сфер (см. список в конце документа)

## 10. Рекомендованные предметы ЕНТ

### 10.1 Таблица пар предметов
- [ ] Логика расчёта:
  1. Берём только **топ-1 сферу** каждого ученика
  2. В этой сфере берём **3 профессии**
  3. Считаем процент по парам предметов
- [ ] Формат таблицы:

| Пара предметов | Процент |
|----------------|---------|
| Математика; Физика | X% |
| Математика; Информатика | X% |
| Математика; География | X% |
| Биология; Химия | X% |
| Биология; География | X% |
| Химия; Физика | X% |
| Всемирная история; Основы права | X% |
| Всемирная история; География | X% |
| Всемирная история; Иностранный язык | X% |
| География; Иностранный язык | X% |
| Казахский/русский язык; Казахская/русская литература | X% |
| Творческий экзамен; Творческий экзамен | X% |

### 10.2 Топ-5 рекомендованных предметов ЕНТ
- [ ] Логика расчёта:
  1. Распарсить пары предметов из таблицы выше
  2. Посчитать частоту каждого отдельного предмета
  3. **Творческий экзамен делится пополам** (потому что задвоен в паре)
  4. Вывести топ-5 наиболее встречающихся
- [ ] Формат: нумерованный список
