Генерация резюме пользователя.
На вход модель получает
1. внеурочную деятельность школьника 
2. триплет RIASEC 
3. трио топ-софтов 
4. три супер-силы по high5
5. три сферы с reasoning
Управляющий промпт: report_resume_prompt.json
Пример реализации функции resume_generation.py
На выходе модель даёт 2 абзаца текста, которые перегоняются на язык пользователя через translate.py
